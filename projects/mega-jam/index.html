Epic MegaJam Submission

A one-week game jam build: a room-based puzzle game with a rewind mechanic that lets players undo mistakes and retry key moments.

Repo
  • UE4
  • C++
  • Gameplay

Demo

Problem

In one week, we had to build a game inspired by the prompt: “It’s been a long time, but we’re not done yet.”

Solution

  • Built a room-based puzzle game where players interact with objects to unlock the exit
  • Implemented a rewind mechanic so players can undo mistakes and retry key moments
  • Created a cohesive set of 3D assets to support each room’s theme and visual language

Code snippet

// This function runs each tick. It verifies whether the player is rewinding.
// If the player is rewinding, it will play the rewind process.
void AMJ20Character::Tick(float DeltaTime)
{
	Super::Tick(DeltaTime);

	FVector TickDeltaMove = (DeltaForward + DeltaRight) * DeltaTime;
	if (!bIsRewinding)
	{
		PlayNextMove(TickDeltaMove, NewRotation);
	}
	else
	{
		PlayNextRewindMove();
	}

	CreateTrace();
}

// If the player is rewinding, this function will play the next location from
// a list of previous locations.
void AMJ20Character::PlayNextRewindMove()
{
	if (RewindProperties.Num() > 0)
	{
		int index = RewindProperties.Num() - 1;
		if (RewindProperties[index].bInteracted)
		{
			UE_LOG(LogTemp, Warning, TEXT("%d"), RewindProperties.Num());
			Interact(RewindProperties[index].InteractedWith);
			RewindProperties[index].bInteracted = false;
		}
		FVector CurrentLocation = GetActorLocation();
		FVector NextDestination = RewindProperties[index].RewindMove.GetLocation();
		FVector DistanceBetweenPoints = NextDestination - CurrentLocation;
		float DistanceToDestination = DistanceBetweenPoints.Size();
		DistanceBetweenPoints.Normalize();

		if(DistanceToDestination > 50.f)
		{
			FVector RewindDeltaMove = DistanceBetweenPoints * RewindSpeed * GetWorld()->DeltaTimeSeconds;
			PlayNextMove(RewindDeltaMove, GetCapsuleComponent()->GetComponentRotation());
		}
		else
		{
			if (RewindProperties[index].bInteracted && RewindProperties[index].InteractedWith != nullptr)
			{
				AMJ20LeverActor* Lever = Cast(RewindProperties[index].InteractedWith);
				if(Lever != nullptr)
				{
					Interact(Lever);
				}
			}
			RewindProperties.RemoveAt(index);
		}
	}
	else
	{
		bIsRewinding = false;
		GetCapsuleComponent()->SetCollisionEnabled(ECollisionEnabled::QueryOnly);
	}
}

What I learned

A tight deadline pushed me to plan like a small production: define a minimal feature set, build in increments, and refactor toward a stable structure. I got better at isolating responsibilities, keeping logic predictable, and avoiding “quick fixes” that create long-term maintenance problems.
