IntConBasMMO

A project where I built game mechanics for spawning and gathering resources, crafting, and basic game combat.

Repo Live Demo
  • C++

Resource Spawn Demo

Problem

For a game where a player can gather resources, a reliable and logical spawning of those resources is necessary, I needed to create that mechanic.

Solution

  • Created a volume within which resources will spawn at terrain level
  • Created a minimum distance between the resources
  • Persistence over state

Code snippet

// This method returns the Fueling Positions that are actively fueling from a table in the database.
@Override
public List<Integer> getActiveFuelingPositionById() {
  return activeTransactionRepo.findAll().stream()
    .map(ActiveTransaction::getFuelingPosition)
    .toList();
}

// This method returns fueling positions that are not currently active, and are available to the equipment whose number has been entered.
@Override
public List<Integer> getInactiveFuelingPositionById(int trId) {
  System.out.println();
  try {
    return fuelingPositionRepo
      .findAll()
      .stream()
      .filter(pr -> startupTransactionRepo.findById(trId).orElseThrow().getEquipmentId().getProducts().stream()
        .map(Product::getProductName)
        .anyMatch(pr2 -> pr.getProduct().getProductName().equals(pr2)))
      .map(FuelingPosition::getFuelingPositionNumber)
      .filter(Predicate.not(hoseN -> activeTransactionRepo.findAll().stream()
        .map(ActiveTransaction::getTransactions)
        .map(Transaction::getFuelingPosition)
        .anyMatch(x -> x.equals(hoseN))))
      .toList();
  } catch (Exception e) {
    System.out.println(e);
    return List.of();
  }
}
